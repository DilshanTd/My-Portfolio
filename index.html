Creamy Ice Ui Design(Figma)

Creamy Ice is a mobile application designed for online ice cream delivery, offering an intuitive and visually appealing user interface across seven essential pages. The Register Page allows users to create new accounts with form validation, while the Log In Page provides a simple and secure login process. The Home Page serves as the main dashboard, featuring various ice cream categories, popular products, a search bar, and promotional banners. The Customer Profile page lets users view and edit personal information, delivery addresses, payment methods, and app settings. The Cart Page displays selected items with modification options and a streamlined checkout process. The Purchase History page offers a detailed list of past orders, with options to reorder and search for specific orders. The Product Description Page provides comprehensive information about each ice cream product, including images, ingredients, price, and reviews, with the ability to add items to the cart directly. The artistic design theme complements the vibrant nature of the product, ensuring an engaging and enjoyable user experience.
